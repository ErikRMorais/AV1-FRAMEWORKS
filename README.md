# Sistema de Gestão de Compras

**Erik R. Morais**

---

Aplicação desenvolvida com Vue.js 3 para gerenciar uma lista de compras em tempo real.

## Funcionalidades

- Adicionar produtos via botão ou tecla **Enter**
- Remover produtos individualmente
- Exibição condicional da lista (cesta vazia / com itens)
- Contador automático de itens

## Tecnologias

- Vue.js 3 (Composition API com `<script setup>`)
- TypeScript
- Vite

## Como rodar o projeto

```bash
# 1. Entrar na pasta do projeto
cd AV01

# 2. Instalar as dependências
npm install

# 3. Iniciar o servidor de desenvolvimento
npm run dev
```

Acesse em: [http://localhost:5173](http://localhost:5173)
